## Recording Tips

1. **Screen recording:** Use OBS or QuickTime
2. **Resolution:** 1920x1080 minimum
3. **Audio:** Clear microphone, minimal background noise
4. **Pace:** Speak slowly, pause between sections
5. **Editing:** Cut dead air, add captions if needed
6. **Length:** Aim for 2:30, max 3:00
7. **Upload:** YouTube (unlisted), Loom, or direct file
